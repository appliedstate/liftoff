# Arbitrage — Onboarding in Four Hours

Agenda (4 hours)
1) Rules (30m): review `01-rules-of-autonomy.md`; sign‑to‑work acknowledgement.
2) Purpose & KPIs (30m): `00-purpose-and-scope.md`, `04-kpis-as-experiments.md`.
3) DSM & Tools (60m): Strateg.is dashboards, curated API, MCP tools; freshness SLAs; evidence links.
4) Operations (60m): kill/keep/scale rules; risk & comp guardrails; receipts/cash cushion model.
5) Hands‑on (60m): launch a small canary; attach evidence to Experiment Board; EOD reconciliation.

Access checklist
- Strateg.is account (dashboards + terminal read permissions)
- API read role credentials; OpenAPI schema link
- Repo/docs access; change notes templates; Experiment Board link

Acceptance
- Complete sign‑to‑work
- Demonstrate a canary launch with proper guardrails and evidence links
- Answer a receipts/cushion scenario correctly
