# Creative Ops Checklist — Catherine (DRI)

Intake
- Receive approved Blueprint + Creative Batch Spec
- Clarify audience, offer, thumb‑stop angle, formats, deadlines

Production
- Produce 5–8 assets per batch; prioritize thumb‑stops and first‑3‑seconds hook
- Apply spec: sizes, durations, captions, safe‑zones, brand guidelines
- Versioning and naming conventions; maintain a simple changelog

Export & QA
- Export presets per platform; verify file size/bitrate; no artifacting
- Spell‑check, legibility, sound mix; compliance checks

Delivery
- Package assets with manifest; include thumbnails and copy snippets
- Handoff to Maryna with due dates; confirm receipt

Iteration Loop
- Review performance signals daily with Dan/Maryna
- Propose and ship iterative variants within 24h of feedback

Documentation
- Record Looms for best‑performing patterns and editing recipes
- Store assets and manifests in the shared library; link in checklist
