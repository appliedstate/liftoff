# Batch 02 Progress Review — Meaningful Change 3

This packet is designed for operator review in under two minutes.

## Change 3

### Change

Built the buyer opportunity mix layer directly into the canonical buyer scorecard so each buyer now carries owned-opportunity inventory, pending/stale pressure, predicted upside still sitting upstream, and source/category concentration.

### Responsible Board Seat

`Warren Buffett`

Why this seat:

This change is about judging performance in context. A buyer’s result should not be evaluated as raw economics alone if the portfolio mix itself is meaningfully different.

### Why It Needs To Exist

Before this change, the scorecard could show:

- economics
- attribution confidence
- execution pressure

But it could not show:

- what kind of opportunity inventory a buyer actually owns
- whether that inventory is stale
- whether upside is still stuck upstream
- whether the buyer’s opportunity mix is concentrated in one source or category

### Why Now

Batch 02 is specifically about grounding buyer performance to the scorecard.

Once attribution was tightened, the next truth problem was mix:

the system still could not separate buyer quality from portfolio quality.

### Limiting Factor It Tackles

Primary limiting factor:

`buyer performance was visible, but buyer opportunity mix was still opaque`

More specifically:

the operator could see output, but not enough of the upstream opportunity context needed to judge whether that output reflected durable buyer edge or an easy inherited portfolio.

### Operator Read

This change should let the operator answer:

1. How many opportunities does each buyer actually own?
2. How many of those are still pending or stale?
3. How much predicted upside is still trapped upstream?
4. Is the buyer’s opportunity mix overly concentrated in one source or category?
