## 02 — Ad Script

### Policy filter (must pass)

- [ ] No diagnosis/personal attributes (“you have…”, “your condition…”)
- [ ] No guarantees (“will”, “guaranteed”, “instant”, “cure”)
- [ ] No claims the **article cannot support**
- [ ] CTA is informational (“see options”, “learn more”, “see what’s included”)
- [ ] Disclosures included where required (“Not medical advice”, “Results vary”, etc.)

### Copy persona + scoring (required)

- **Persona**: *The Calm Contrarian Operator* (see `docs/edge/workflows/ad-copy-framework.md`)
- **Score each concept (0–10)** using the rubric; ship only **≥ 8/10**.
