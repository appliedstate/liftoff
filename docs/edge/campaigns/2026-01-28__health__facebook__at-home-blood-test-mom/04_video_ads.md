## 04 — Video Ads (production briefs)

### Shared production specs

- **Length**: 8–10s (all)
- **Format**: 9:16 primary; export 4:5 and 1:1
- **Style**: UGC selfie + screen overlay
- **Mandatory overlay**: article headline (or shortened)
- **Disclosure**: add tiny footer text (“For informational purposes”) only if required; avoid putting disclaimers in headlines.
- **Mandatory proof**: show the 3–4 RSOC widget buttons on screen for 1–2s

### Visual convention (so it looks like “a thing”)

- Frame 1: BIG overlay (pattern interrupt)
- Frame 2: headline (matches article)
- Frame 3: RSOC widget buttons (3–4 phrases)
- Frame 4: CTA (“Tap to see the checklist”)
